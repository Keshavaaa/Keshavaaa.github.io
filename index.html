aLMARi - AI-Powered Digital Wardrobe

React Native • Google Gemini API • SQLite • Computer Vision

Full-stack mobile application that helps users organize their wardrobe with AI-powered clothing analysis, outfit recommendations, and weekly planning. Built to solve my own daily clothing selection confusion, this app demonstrates end-to-end mobile development with advanced AI integration.

Core Features Built:

  • AI clothing recognition using Google's Gemini API with custom prompt engineering
  • Automatic background removal and image compression pipeline
  • Color extraction using ColorThief library for smart categorization
  • Outfit calendar with weather-aware scheduling and recommendations
  • Laundry tracking system with favorites and usage analytics
  • Local SQLite database with optimized storage for privacy-first design

Technical Challenges Overcome: Resolved SQLite CursorWindow overflow errors through base64 optimization, implemented multi-stage image processing workflow (removal → compression → analysis), designed complete React Native architecture with Expo Router navigation, and integrated real-time AI processing.

PDF Assistant - AI Powered Learning Tool

Python • LangChain • Streamlit • Google Gemini API • RAG Pipeline

Intelligent learning assistant that transforms static PDF documents into interactive study tools. Inspired by the vision of "books that can talk," this application uses LangChain and Gemini AI to enable conversational learning from any PDF document.

Core Features Built:

  • Conversational Q&A based on PDF content with context retention
  • Automated MCQ generation for self-assessment and practice
  • Personalized weekly study schedule creation based on document analysis
  • Document summarization for quick content overview
  • Complete RAG (Retrieval-Augmented Generation) pipeline implementation

Technologies: Python, Streamlit, LangChain, PyPDF2, Google Gemini API, prompt engineering, document chunking strategies

Technical Highlights: Developed specialized prompts for three distinct functions (Q&A, MCQ generation, study planning), implemented efficient document processing and chunking, built complete RAG pipeline from scratch using LangChain framework, and designed user-friendly Streamlit interface for seamless interaction.
