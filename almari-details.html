aLMARi - AI-Powered Digital Wardrobe

A full-stack mobile application built from the ground up to solve the daily challenge of choosing an outfit. This project showcases end-to-end development, from UI/UX design in React Native to a complex, privacy-first AI backend running on-device.

1. Professional Onboarding

A clean, animated splash screen provides a professional first impression, leading into a simple onboarding flow that guides the user.

2. AI-Powered Clothing Analysis

Users can upload a photo of a clothing item. The app automatically removes the background, extracts dominant colors using ColorThief, and sends the processed image to the Google Gemini API for categorization and description, all orchestrated through custom prompt engineering.

3. Interactive Wardrobe

All clothing items are displayed in a clean grid. Users can filter by category, color, or status (e.g., in laundry). All data is stored locally in an SQLite database for fast, offline, and private access.

4. Detailed Item Management

Each item has a detailed screen showing its category, color, description, and usage history. From here, users can add items to the laundry, edit details, or delete them from their wardrobe.

5. Smart Outfit Recommendations

The home page dashboard features AI-generated outfit suggestions. The model considers the user's existing wardrobe and the local weather to propose a suitable and stylish combination for the day.

6. Outfit Scheduling Calendar

A fully functional calendar allows users to plan their outfits for days or weeks in advance, helping to streamline their morning routine. Scheduled outfits are displayed visually on the calendar.

7. User Profile and Analytics

The profile screen gives users an overview of their wardrobe statistics, such as the total number of items, most-worn pieces, and items currently in the laundry, providing useful insights into their clothing usage.
